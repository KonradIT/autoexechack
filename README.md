autoexec hack
============

YEAH! Lets hack GoPro cameras using autoexec!

Instructions:
-------------
Using a proper text editor (notepad?), experiment with commands avaible in this repository to tell the camera what to do.
Save the file as "autoexec.ash" (.ash ending!)
